## Proyectos

### TabMemFree
2011 - actualidad

https://chromewebstore.google.com/detail/tabmemfree/pdanbocphccpmidkhloklnlfplehiikb

Por aquel entonces tenía un portátil con muy poca memoria RAM. Cada vez que cambiaba de aplicación tardaba una eternidad debido al intercambio de datos con el disco duro. Y Chrome era el principal responsable de este problema.

Así que creé una extensión para Chrome. Reducía el consumo de RAM y la carga de la CPU «aparcando» las pestañas inactivas en una página vacía, al tiempo que conservaba el título y el icono de la pestaña. Al activar la pestaña, «retrocedía» en el historial para restaurar el estado de la pestaña casi al instante. Fue la primera extensión de este tipo para Chrome, que presenté en una conferencia de Google.

Años más tarde, Chrome proporcionó una API para «suspender» pestañas sin perder el estado ni afectar al historial. Y, tras una automatización de la suspensión, TabMemFree quedó obsoleta para siempre.

### Sistema de almacenamiento de LEGO
2025

#### Problema
Mi hijo tenía un montón de piezas de LEGO repartidas equitativamente en 5 cajas Trofast de IKEA. Mientras construía, dedicaba la mayor parte del tiempo a buscar piezas.

#### Solución
Para solucionarlo, creé un sistema de almacenamiento de piezas de LEGO:
- analicé el inventario de piezas
- definí grupos de piezas distintos
- dividí los grupos por tamaño, si este variaba mucho
- realicé pruebas para verificar y ajustar la lógica de agrupación según las expectativas del usuario final
- elegí tamaños de separadores imprimibles en 3D para las cajas Trofast de IKEA, que pudieran albergar los grupos, ocupando el menor espacio posible y permitiendo localizar una pieza dentro de un grupo en un par de segundos
- optimicé los ajustes de impresión 3D para reducir el tiempo de impresión y el material utilizado, al tiempo que se garantizaba la resistencia suficiente
- realicé sesiones de montaje y desmontaje asistido de modelos de LEGO para verificar la integridad del sistema bajo una carga elevada

#### Resultados
- Tiempo de acceso 10 veces más rápido
- Ruido de búsqueda 3 veces menor
- 5 veces más piezas a la venta en la tienda
